\section{Analítica de Datos}
\begin{frame}
  \begin{block}{Definición}
La analítica es el proceso de descubrir, interpretar y comunicar patrones significativos en los datos. . En pocas palabras, la analítica nos ayuda a ver información y datos significativos que de otro modo no podríamos detectar. La analítica empresarial se centra en el uso de conocimientos derivados de los datos para tomar decisiones más informadas que ayudarán a las organizaciones a aumentar las ventas, reducir los costes y realizar otras mejoras comerciales.
   \end{block}
\[
\includegraphics[scale=0.15]{gooble-analytics.png}
\]
\end{frame}
\begin{frame}
	\begin{block}{Usos}
		Las organizaciones pueden aplicar análisis a los datos comerciales para describir, predecir y mejorar el desempeño comercial.
		\begin{itemize}
			\item análisis predictivo
			\item análisis prescriptivo
			\item gestión de decisiones empresariales
			\item análisis descriptivo
			\item análisis de Big Data
			\item optimización de unidades de almacenamiento y surtido de tiendas
			\item optimización de marketing y modelado de mezcla de marketing
			\item web análisis
			\item análisis de llamadas
			\item análisis de voz
			\item modelado de precios y promociones
			\item análisis de riesgo crediticio
			\item análisis de fraude.
		\end{itemize}
	\end{block}
\end{frame}

\begin{frame}
	\begin{block}{Tipos de análisis de datos}
		¿Qué tan profundo se debe llegar a los datos en busca de ideas o hipótesis basadas en hechos?
		
		Hay 4 tipos diferentes de análisis. 
		\begin{itemize}
			\item \textbf{Analítica descriptiva}
			Técnica que responde a la pregunta de qué sucedió.
			\item \textbf{Analítica diagnóstica}
			En esta etapa, los datos históricos se pueden contrastar con otros datos para responder a la pregunta de por qué sucedió algo. 
			\item \textbf{Analítica predictiva}
			Técnica que se anticipa y sugiere lo que es probable que suceda. Utiliza los hallazgos de análisis descriptivos y de diagnóstico para detectar grupos y excepciones, y para predecir tendencias futuras, lo que lo convierte en una herramienta valiosa para la predicción.  
		\end{itemize}
	\end{block}
\end{frame}
\begin{frame}
	\begin{block}{}
		\begin{itemize}
			\item \textbf{Analítica prescriptiva}
			.Es prescribir qué medidas tomar para eliminar un problema futuro o aprovechar al máximo una tendencia. Este tipo de análisis requiere no sólo datos internos históricos sino también información externa debido a la naturaleza de los algoritmos en los que se basa.
		\end{itemize}
	\end{block}
	\begin{block}{Beneficios para las empresas}
	\begin{itemize}
		\item El uso de analítica de datos reduce lo costos
		\item El uso de analítica de datos aumenta la eficiencia
		\item El uso de analítica de datos mejora los precios
	\end{itemize}
\end{block}
\end{frame}
